Jules Debug Commands

Quick reference snippets to run directly in your browser's DevTools console.

1. Verify Bridge Connection

Checks if the context bridge is loaded and dumps the exposed Electron APIs.

console.log({
  isElectron: !!window.electron?.window,
  bridgeAPI: window.electron
});

2. Load Saved Aliases

Reads your loaded aliases.json list directly from the bridge and displays them in a clean table.

window.electron.aliases.get().then(res => {
  console.log("%cSaved Aliases List:", 'font-weight: bold; color: #3b82f6;');
  console.table(res.aliases);
});

3. Retrieve Session Info

Fetches live metadata (branch, repo, status, timestamp) of the active testing session.

window.electron.sdkIpc.resumeSession("4021342359891090888")
  .then(info => console.log("%c✓ Session Info:", 'color: #10b981; font-weight: bold;', info))
  .catch(err => console.error("Failed to resume session:", err));

4. Fetch Session Files (Markdown)

Queries the session database for generated markdown logs and dumps their contents to the console.

window.electron.sdkIpc.getMarkdownFiles("4021342359891090888")
  .then(files => {
    console.log(`%c✓ Found ${files.length} markdown files:`, 'color: #10b981; font-weight: bold;');
    files.forEach(f => {
      console.log(`%c[${f.path}]`, 'color: #8b5cf6; font-weight: bold;');
      console.log(f.content);
    });
  })
  .catch(err => console.error("Failed to fetch files:", err));

5. Send Prompt to Jules

Sends a test prompt message directly to the session running on Google's API.

window.electron.sdkIpc.sendMessage("4021342359891090888", "Say hello in markdown please.")
  .then(() => console.log("%c✓ Message sent successfully!", 'color: #10b981; font-weight: bold;'))
  .catch(err => console.error("Send failed:", err));
